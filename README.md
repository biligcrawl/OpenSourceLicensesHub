## Installation
*__for Linux/Debian/Ubuntu__*
- Do the following operations in order.


### System Update & Upgrade
```bash
sudo apt update
```
```bash
sudo apt upgrade
```

### Clone Repository
```bash
git clone https://github.com/biligcrawl/OpenSourceLicensesHub
```

- Change to the project directory:
```bash
cd OpenSourceLicensesHub
```
